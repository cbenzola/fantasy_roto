User Functionality
-------------------------------------------------------------------------------------

- For user functionality of diamond, lineup builder, register, login and player searches --
  make sure you are either on fantasyroto.com or connected to a localhost.


- The image maps I put on the trending players on the homepage lead
  to the stat pages for those players, so you will need to be on the website or
  connected to a localhost

- To use the Lineup Builder you must be a registered user

- The Lineup Builder should be working(hopefully)

Steps to use Lineup Builder
Step 1: Login
Step 2: Press Clear
Step 3: Select one P,C,1B,2B,3B,SS and three OF
Step 4: Press Save
Step 5: Press Optimize
